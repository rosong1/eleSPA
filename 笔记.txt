//   vue-resource应用
1.在main.js先import进去，再用Vue.use全局注册
2.在父组件(App.vue)中使用 
   data() {
     return {
       seller:{}
  }
}

3.接着在父组件中运用created属性，分状态码返回。返回对象用：response.body
4.父组件引用子组件的地方用v-bind绑定 <v-header :seller="seller"></v-header>
5.子组件中用props属性声明要绑定的json数据对象（seller）,声明type:Object
